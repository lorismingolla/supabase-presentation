Key Features

  • Database
  • Auth
  • Storage
  • Edge Functions
  • Realtime
  • AI & Vectors

Database

PostgreSQL for a robust, scalable database with support for:

  • Complex queries and transactions
  • Full-text search
  • Extensions like PostGIS
  • Supabase Row-Level Security for fine-grained access control

Authentication

Comprehensive authentication system that includes:

  • Email/password sign-ups and logins
  • Third-party logins with OAuth providers (e.g., Google, GitHub)
  • Magic links for passwordless authentication
  • Leans on the database

Storage

File storage solution integrated with your database:

  • Bucket creation and management
  • File upload and access controls
  • Public and private storage options
  • Integration with PostgreSQL for metadata management

APIs

Autogenerated RESTful and GraphQL APIs based on your database schema, allowing you to:

  • Perform CRUD operations on your database
  • Secure endpoints with row-level security
  • Subscribe to real-time changes via WebSockets

Edge Functions

Serverless functions at the edge, providing:

  • Seamless integration with Supabase services
  • Ability to extend functionality beyond the database
  • Integration with HTTP APIs and database triggers
  • All stuff you don't want to handle on the client-side..

Realtime

Supabase supports real-time functionality, offering:

  • Broadcasting of database changes
  • Real-time subscriptions
  • Presence tracking

AI and Vectors

Supabase supports AI and vector operations, providing:

  • Storage and management of embeddings
  • Vector similarity search for AI applications
  • Integration with popular AI frameworks and tools
  • Use of PostgreSQL extensions like pgvector for vector storage and search
